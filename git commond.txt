Github

Local -> pc
Globle -> Github

default branch -> master

--> first-time upload 
1. git init
2. git status
3. git add .
4. git commit -m "commit_message"
5. git remote add origin __URL__
6. git push origin master


--> clone (fetch)
1. git clone __URL__


--> Re-upload
1. git status
2. git add .
3. git commit -m "commit_message"
4. git push origin master


--> fetch server data
1. git fetch
2. git pull origin master


-> change branch
git branch -m branch_name
